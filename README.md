# 全国預金キャンペーン Web版

`campaign_all.json` を外部データソースとして読み込み、全国の預金キャンペーンを検索・表示するローカルWebアプリです。外部CDNや外部ライブラリは使用していません。

## ファイル構成

- `index.html`：画面本体
- `styles.css`：デザイン・レスポンシブ表示
- `app.js`：JSON読込、絞り込み、集計、金利ランキング、ガントチャート、表、CSV出力
- `campaign_all.json`：全国データ
- `run_web_windows.bat`：Windows用起動ファイル
- `run_web_mac.sh`：Mac用起動ファイル
- `README.md`：本書

## Windowsでの起動

1. すべてのファイルを同じフォルダーに置きます。
2. Pythonがインストールされていることを確認します。
3. `run_web_windows.bat` をダブルクリックします。
4. ブラウザーで `http://localhost:8000/` が開きます。
5. 終了するときは、コマンドプロンプトで `Ctrl + C` を押してウィンドウを閉じます。

`py`、`python`、`python3` の順で利用可能なコマンドを自動判定します。

## Macでの起動

ターミナルで、このフォルダーへ移動して次を実行します。

```bash
chmod +x run_web_mac.sh
./run_web_mac.sh
```

ブラウザーで `http://localhost:8000/` が開きます。終了するときはターミナルで `Control + C` を押します。

## index.htmlを直接開かない理由

`file://` で `index.html` を直接開くと、ブラウザーのセキュリティ制限により `campaign_all.json` の読み込みが拒否されることがあります。必ず同梱の起動ファイル、または次の簡易HTTPサーバーを使用してください。

```bash
python -m http.server 8000
```

Macなどで `python` がない場合は次を使用します。

```bash
python3 -m http.server 8000
```

## 操作方法

### 初期表示

描画負荷を抑えるため、初期状態は「開催中＋開催予定」のみを表示します。全国全件を表示する場合は「条件をクリア」を押してください。

### 絞り込み

地域、都道府県、金融機関種別、商品区分、開催状況は複数選択できます。Windowsでは `Ctrl`、Macでは `Command` を押しながら選択してください。

金融機関名、キャンペーン名、年限、金利は部分一致検索です。開始日・終了日はそれぞれ「以上」「以下」で範囲指定できます。

#### 直近期間のクイック設定

「直近6か月」「直近1年」「直近3年」「直近5年」のボタンを押すと、ブラウザーを開いた日を基準に日付条件を自動設定します。

- キャンペーン開始日「以上」：今日から6か月・1年・3年・5年前の日付
- キャンペーン終了日「以下」：今日
- 開始日「以下」と終了日「以上」：空欄

たとえば2026-07-19に「直近3年」を押すと、開始日「以上」に2023-07-19、終了日「以下」に2026-07-19が入ります。指定期間内に開始し、今日までに終了したキャンペーンを抽出します。終了日未設定のキャンペーンは継続中として検索対象に含めます。過去の終了済みキャンペーンも表示できるよう、期間ボタンを押した時点で開催状況フィルターは自動的に全件対象へ切り替わります。「期間指定を解除」を押すと、4つの日付欄だけを空欄に戻します。ボタンで設定した後に日付を手入力で変更した場合は、通常の開始日・終了日範囲検索へ切り替わります。

### 想定満期年検索

「想定満期年」で年を選ぶと、その年に満期が到来すると推定される明細を抽出します。過去の終了済みキャンペーンも検索対象にするため、満期年を選択した時点で開催状況フィルターは自動的に全件対象へ切り替わります。その後、必要に応じて開催状況を再選択できます。

想定満期期間は、キャンペーン期間中に預入したと仮定し、次のように算出します。

- 最も早い満期：キャンペーン開始日＋預入期間・年限
- 最も遅い満期：キャンペーン終了日＋預入期間・年限
- 終了日未設定：開始日＋預入期間・年限のみ
- 年限が範囲表記：最短年限から最長年限までを推定範囲として使用
- 開始日不明、または年限を機械判定できない表記：満期年検索の対象外

たとえば、2023年に募集された3年定期は、募集期間に応じて2026年の想定満期として抽出されます。実際の満期日は実際の預入日、自動継続、中途解約、商品固有条件などで異なります。

### 期間別金利ランキング・マトリクス

「対象キャンペーン期間」「地域」「商品区分」を選ぶと、6か月・1年・3年・5年の期間別に最高金利、平均金利、上位5金融機関を表示します。対象期間は、キャンペーン募集期間と1日以上重なる明細を採用します。

- 最高：期間区分ごとに、比較可能な金融機関別金利の最高値と該当金融機関名を平均の上段に表示
- 平均：同一金融機関に複数明細がある場合、期間区分ごとの最高金利を1件として平均
- ランキング：期間区分ごとに金融機関単位で上位5を抽出
- マトリクス：X軸が6か月・1年・3年・5年、Y軸がいずれかの期間で上位5に入った金融機関、セルが適用金利
  - 期間ごとに上位金融機関が異なるため、Y軸は最大20金融機関になる場合があります。各セルの「1位～5位」バッジで期間別順位を確認できます。
- 商品区分を複数選択した場合：定期預金、定期貯金、積金などの商品区分ごとに別マトリクスを生成
- 金利改定履歴の「←」：左端の最新金利を使用
- 店頭金利への上乗せ式・倍率式：最終的な絶対金利が明示されていない場合は平均・ランキングから除外
- 年限が範囲・複数・判定不能の明細：固定4期間のマトリクス対象外

初期値は当年1月1日から12月31日、商品区分は「定期預金」、地域は全国です。

### ガントチャート

- Y軸：金融機関・キャンペーン
- X軸：キャンペーン期間
- 終了日未設定：斜線・矢印付きで表示範囲右端まで表示
- 日付逆転：赤いマーカーで表示
- バーにマウスを合わせる：詳細ツールチップ
- バーをクリック：対応する明細のみを下表に表示

金融機関は10・20・30件単位でページ送りできます。同一キャンペーン・同一期間の複数年限や複数金利は1本のバーにまとめ、ツールチップ内に列挙します。

### 日付要確認

開始日不明、終了日未設定・不明、終了日が開始日より前、日付関連の要確認事項を持つ明細を一覧表示します。

### 明細一覧

50・100・200件単位でページ送りできます。列見出しをクリックすると昇順・降順を切り替えます。「想定満期期間」列には推定可能な期間を表示し、推定できない行は「推定不可」と表示します。商品URLがある行は新しいタブで開きます。

### CSVダウンロード

「絞り込み結果をCSVダウンロード」を押すと、現在の絞り込み条件に一致する全明細をUTF-8 BOM付きCSVで保存します。CSVには計算列として「想定満期開始日」「想定満期終了日」「満期推定注記」も追加されます。ガントチャートで選択した一時的な明細フォーカスはCSVの対象を狭めません。

## エラー表示

- `campaign_all.json` がない
- JSON形式が不正
- `records` 配列がない
- `metadata.record_count` と実件数が不一致

上記の場合、画面上部に日本語のエラーまたは警告を表示します。データ0件でも各領域は空表示となり、画面は壊れません。

## データ更新

新しい全国版JSONを使用する場合は、ファイル名を `campaign_all.json` として上書きしてください。スキーマのキー名は現在のJSONと同一にしてください。
